# 社区行为准则 / Community Code of Conduct

MineM 致力于建设相互尊重、专业且不受骚扰的社区。

MineM is committed to a respectful, professional, and harassment-free
community.

贡献者必须 / Contributors must:

- 就想法和代码展开讨论，不进行人身攻击；discuss ideas and code without
  personal attacks;
- 尊重不同背景、经验水平和观点；respect different backgrounds, experience
  levels, and viewpoints;
- 提供可执行的反馈，并善意接受纠正；give actionable feedback and accept
  correction in good faith;
- 不发布私人信息、客户数据或凭据；avoid publishing private information,
  customer data, or credentials;
- 遵守维护者对项目范围和社区安全的决定；follow maintainer decisions about
  project scope and community safety.

社区不接受骚扰、歧视、威胁、性化行为、故意侵犯隐私或持续干扰。

Harassment, discrimination, threats, sexualized conduct, deliberate privacy
violations, and sustained disruption are not accepted.

请通过仓库的 Security 报告渠道私下反馈行为问题。为保护社区，维护者可以编辑或
删除内容、关闭讨论、拒绝贡献，或临时及永久限制参与。

Report conduct concerns privately through the repository's Security reporting
channel. Maintainers may edit or remove content, close discussions, reject
contributions, or temporarily or permanently restrict participation when
needed to protect the community.
